1. 筛选素数

   筛选的素数这个就是根据埃拉托斯特尼筛法筛选，即要得到自然数n以内的全部素数，必须把不大于√n的所有素数剔除，剩下的就是素数。

2. CSS 计数器及其作用域

   1. counter-reset
   2. counter-increment

3. 伪元素选择器与伪类选择器

   1. :before
   2. :fisrt-child
   3. :nth-child
